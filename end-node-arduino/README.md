### Functions

- Weight monitoring

- Arduino (UNO/Nano)
- DHT11 Sensor
- MQ2 Gas Sensor
- HX711 Load Cell
- Relay Module
- Motor Driver

### Communication
- BLE (HM-10) to Edge Node
